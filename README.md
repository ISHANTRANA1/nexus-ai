# NexusAI — AI Data Automation Platform

Premium SaaS landing page built for the hackathon Round 1 challenge.

## Tech Stack
- **Framework:** Next.js 16 (App Router)
- **Styling:** Tailwind CSS + Custom CSS Variables
- **Fonts:** JetBrains Mono + Inter (Google Fonts)
- **Animations:** Native CSS Transitions + WAAPI only

## Features Built
- ✅ Feature 1: Multi-currency pricing matrix (INR/USD/EUR × Monthly/Annual) with isolated DOM updates
- ✅ Feature 2: Bento Grid → Accordion with active index context transfer on resize
- ✅ Semantic HTML (header, nav, main, section, article, footer)
- ✅ Full OG + Twitter meta tags
- ✅ Sub-500ms loader animation
- ✅ Zero external UI/animation libraries
- ✅ Fully responsive (mobile/tablet/desktop)
